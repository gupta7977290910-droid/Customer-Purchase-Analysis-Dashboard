# 📊 Customer Purchase Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Customer Purchase Analysis Dashboard built using Microsoft Power BI. The dashboard provides insights into customer purchasing behavior, product performance, payment methods, and sales trends to support data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze customer purchase history.
- Monitor overall sales performance.
- Identify top-selling products.
- Analyze payment method preferences.
- Track monthly sales trends.
- Build an interactive dashboard using Power BI.

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel Dataset
- Data Modeling (Star Schema)

## 📂 Dataset Information

The dataset contains customer purchase records with the following fields:

- Customer ID
- Customer Name
- Product
- Product Category
- Purchase Date
- Quantity
- Unit Price
- Total Price
- Payment Method
- Review Rating

---

## 📈 Dashboard Features

### KPI Cards
- Total Sales
- Total Customers
- Total Orders
- Average Customer Rating

### Interactive Visualizations
- Sales by Product Category
- Monthly Sales Trend
- Orders by Payment Method
- Product-wise Sales (Treemap)

### Filters (Slicers)
- Purchase Date
- Payment Method
- Product Category

---

## 📊 Power BI Concepts Used

- Data Cleaning using Power Query
- Data Modeling
- One-to-Many Relationships
- Star Schema
- DAX Measures
- Interactive Slicers
- Card Visuals
- Bar Chart
- Donut Chart
- Treemap
- Line Chart

## 📌 Key Insights

- Electronics generated the highest sales.
- Credit Card was the most preferred payment method.
- Monthly sales trends help identify seasonal performance.
- KPI cards provide a quick overview of business performance.
